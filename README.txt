Description:
This is login based application. It will accept only two users account with username
abe or barb and password is password. If any other username or password is entered
it will show a message "Username or password is incorrect". Once successfully logging
in, home page shows up and with a message "Hello (name of the user)", and a link to 
logout from the session with a logout message, which will bring back to login page.
Once the user is logged in , it can only go back to login page through logout option
in home page. A user can only access home page through logging in with the credentials
mentioned above. No other username or password is valid with this application.
